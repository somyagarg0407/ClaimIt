# 🚀 Overview

ClaimIt is a modern GovTech platform built to simplify access to government welfare programs.

Instead of manually browsing hundreds of government portals, users can create a profile and instantly discover schemes, scholarships, subsidies, pensions, and welfare programs they may be eligible for.

The long-term vision is to make government benefits easier to discover, easier to understand, and easier to claim through an intuitive AI-assisted experience that feels like a modern SaaS product rather than a traditional government website.

# 👨‍💻 About This Project

ClaimIt is my flagship solo project.

I am the sole developer and product designer behind the project, responsible for the product vision, user experience, frontend architecture, technical decisions, and long-term roadmap.

The frontend follows an AI-assisted development workflow, where every feature is manually reviewed, refined, tested, and integrated to maintain production-quality standards.

The backend is being built entirely from scratch using Node.js, Express.js, MongoDB, and JWT Authentication, with a strong focus on backend engineering, API design, authentication, and scalable application architecture.
